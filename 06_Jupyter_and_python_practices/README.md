# Khanlab Tutorial 6: Jupyter Notebooks and Python Best Practices

## Objective:
Learn about Python virtual environments, Jupyter, how to use it on Graham, and best Python programming practices!

    * Python3 installed

* What is Jupyter
* How to use Jupyter on Graham
